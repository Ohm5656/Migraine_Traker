Design a modern mobile application UI for an AI-powered migraine prevention system called **NeuroGuard**.

The application is primarily designed for **mobile devices** and must follow **clean, structured, component-based layouts that can be easily implemented in Flutter**.

Design Style:

* Dark tone luxury health-tech aesthetic
* Minimal, elegant, premium interface
* Soft gradients, deep blacks, dark blues, and subtle purple accents
* Calm and professional feeling suitable for a health application
* High contrast and accessible typography
* Rounded cards and smooth spacing
* Simple and intuitive user experience

Color Palette:

* Background: deep charcoal / near-black
* Primary accent: electric purple or indigo
* Secondary accent: soft blue
* Status colors:

  * Green (low risk)
  * Yellow (moderate risk)
  * Red (high migraine risk)

Typography:

* Modern sans-serif style
* Clear hierarchy for headings, metrics, and descriptions

Design the following **mobile screens**:

1. **Onboarding Screen**
   Explain the purpose of the app:
   "AI-powered migraine risk prediction and prevention"
   Include simple illustrations of wearable sensors and brain health.

2. **Dashboard Screen**
   Main overview showing:

* Migraine Risk Score (large circular indicator)
* Current stress level
* Sleep quality
* HRV indicator
* Quick summary of health signals
* Clean card layout

3. **Migraine Alert Screen**
   A warning UI when migraine risk is rising:

* Clear warning message
* Risk percentage
* Recommended actions:

  * take a break
  * drink water
  * reduce screen brightness
  * breathing exercise

4. **Trigger Analysis Screen**
   Show personalized migraine triggers detected by AI:

* Sleep deprivation
* High stress
* Bright light exposure
  Use visual charts and ranking cards.

5. **History & Insights Screen**
   Show migraine patterns over time:

* Weekly graph
* Risk trends
* Episode tracking

6. **Device Connection Screen**
   Show connection to a lightweight wearable migraine sensor band.

Layout requirements:

* Use card-based UI
* Mobile-friendly spacing
* Use reusable components
* Avoid overly complex effects so it can be implemented easily in Flutter

The interface should feel:
calm, intelligent, futuristic, premium, and trustworthy.
